<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4 text-center">Quick start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8 text-center max-w-2xl mx-auto">
    Get up and running with SlipStream GUI in minutes
  </p>

  <Steps>
    <Step title="Download and install">
      Download the latest release for your platform from the [GitHub releases page](https://github.com/mirzaaghazadeh/SlipStreamGUI/releases/latest). Double-click the installer and follow the prompts.

      <CodeGroup>
      ```bash macOS (Apple Silicon)
      # Download SlipStream-GUI-macOS-ARM64.dmg
      # Open the DMG and drag to Applications
      ```

      ```bash macOS (Intel)
      # Download SlipStream-GUI-macOS-Intel.dmg
      # Open the DMG and drag to Applications
      ```

      ```bash Windows
      # Download SlipStream-GUI-Windows-x64.exe
      # Run the installer
      ```

      ```bash Linux
      # Download SlipStream-GUI-Linux-x64.AppImage
      chmod +x SlipStream-GUI-Linux-x64.AppImage
      ./SlipStream-GUI-Linux-x64.AppImage
      ```
      </CodeGroup>
    </Step>

    <Step title="Configure your server">
      Launch SlipStream GUI and configure your SlipStream server settings:

      - **Domain**: Your SlipStream server domain (e.g., `s.example.com`)
      - **DNS Resolver**: Your DNS server (default: `8.8.8.8:53`)

      <Note>
        Don't have a server yet? Check out [slipstream-rust-deploy](https://github.com/AliRezaBeigy/slipstream-rust-deploy) for one-command server deployment.
      </Note>
    </Step>

    <Step title="Start the VPN">
      Click the **Start VPN** button and wait for the status indicators to show "Running". Your traffic is now securely routed through SlipStream!

      Use the **Test Proxy Connection** button to verify everything is working correctly.
    </Step>
  </Steps>
</div>
